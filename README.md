# Simple Calculator

A simple calculator that performs basic arithmetic operations including addition, subtraction, multiplication, and division.

## Why use it?

This calculator is useful for quick and simple calculations without needing complex tools. Perfect for students, beginners learning programming, and anyone who needs fast arithmetic computations.

## Features

- ✅ **Add** - Add two or more numbers
- ✅ **Subtract** - Subtract one number from another
- ✅ **Multiply** - Multiply numbers together
- ✅ **Divide** - Divide one number by another
- ✅ Clean and user-friendly interface
- ✅ Error handling for invalid inputs
- ✅ Simple and easy-to-understand code
